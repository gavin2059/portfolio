My experiences

 Feb 2020 - Aug 2020

Research Assistant for Dr. Peter J. Varman

I worked with Dr. Varman to implement Fair-EDF, an earliest-deadline-first request scheduler seeking to fulfill the same percentage of requests from different clients. This involved using exploring different linear regression and decision tree models to predict SSD processing times which are used in the algorithm to predict whether put a request in the low priority queue.

Using a regression tree with 4 nodes per branch, we were able to accomplish fast training (0.14s on a training set of 5000+ requests) and predictions (52000/second) with a 15% error on average on a YACSIM simulated SSD receiving requests from a single client.

 Sep 2019 - Dec 2020

Analyst, Rice Undergraduate Investment Fund

I was a member of the Porfolio Review section, which manages and oversees the fund's holdings. During my time with the fund, I worked with the Industrials and Energy sectors on buy and sell pitches, industry overviews, and semesterly reports. In Energy, I protected a return of 44% on the stock CLB by pitching to hold it rather than sell it.

Python Testcase Generator

I wrote a program in Java to generate a concise set of test cases to ensure a correct implementation for Python programs given a set of reference implementations. The program parses a config file, recursively generates a large set of test cases, then uses the greedy 'solution' of the hitting set problem to reduce said set to a significantly smaller set of test cases, while ensuring that the concise set of tests will catch all buggy implementations.
