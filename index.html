My experience

 Feb 2020 - Aug 2020

Research Assistant for Dr. Peter J. Varman

I worked with Dr. Varman to implement Fair-EDF, an earliest-deadline-first request scheduler seeking to fulfill the same percentage of requests from different clients. This involved using exploring different linear regression and decision tree models to predict SSD processing times which are used in the algorithm to predict whether put a request in the low priority queue.

Using a decision tree with 4 nodes per branch, we were able to accomplish fast training (0.14s on a training set of 5000+ requests, 52000 observations/second) with a 15% error on average on a YACSIM simulated SSD receiving requests from a single client.
